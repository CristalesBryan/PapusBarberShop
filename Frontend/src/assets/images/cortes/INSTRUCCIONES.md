# Instrucciones para Agregar Imágenes de Tipos de Corte

## Imágenes Requeridas

Copia las siguientes imágenes desde `Papus BarberShop/resources/img/` a esta carpeta:

1. **Corte de Caballero.png** → `assets/images/cortes/Corte de Caballero.png`
2. **Corte para niño.png** → `assets/images/cortes/Corte para niño.png`
3. **Arreglo de Barba.png** → `assets/images/cortes/Arreglo de Barba.png`
4. **Corte y Barba.png** → `assets/images/cortes/Corte y Barba.png`

## Ubicación de las Imágenes Originales

Las imágenes originales se encuentran en:
```
../Papus BarberShop/resources/img/
```

## Notas

- Las imágenes deben estar en formato PNG o JPG
- Si alguna imagen no se encuentra, se mostrará una imagen por defecto
- Los nombres de archivo deben coincidir exactamente con los especificados
